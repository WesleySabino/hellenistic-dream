# Hellenistic Dream

Aplicação web local-first para acompanhar peso e cintura e recomendar BULK, CUT ou LIVRE ESCOLHA.

## Regras principais
- Sem backend remoto
- Sem login
- Dados salvos apenas localmente no navegador
- Mobile-first
- Público leigo + uso com personal trainer
